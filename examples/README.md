# Example Sheets

These sheets are included so behavior can be inspected visually.

## `sample-ai-icon-crop.png`

Small synthetic input crop that can be used for a fresh-clone smoke test:

```bash
auto-icon-vectorizer examples/sample-ai-icon-crop.png --out-prefix out/sample-icon
```

## `results-overview.png`

Main README image. It shows only the important user-facing result:

```text
input crop -> generated transparent SVG
```

No masks, no internal stages, no candidate comparisons.

## `pipeline-diagram.png`

README diagram showing the full public process:

```text
cropped icon -> foreground mask -> traced SVG -> optional HTML wrapper
```

It is generated by `scripts/generate_readme_diagrams.py` using the actual
vectorizer output from a synthetic noisy crop.

## `output-contract.png`

README diagram explaining why the package exposes both raw SVG and inline SVG
HTML:

- raw SVG is the default CLI output and the portable asset
- HTML is the same SVG wrapped with class names, metadata, and accessibility
  hooks for direct web insertion

## `transparent-backgrounds.png`

README diagram showing that the generated SVG has a transparent background and
can be applied to light, dark, patterned, or gradient surfaces.

## `icon-vectorizer-regression.png`

Generated by:

```bash
auto-icon-vectorizer-regression
```

It checks that:

- obvious hybrid fill+stroke cases route to the filled branch
- pure outline icons route to the stroke branch
- a hybrid case where stroke is already sufficient is accepted by quality

## `hybrid-fill-stroke-eval-after-auto-threshold.png`

Stress sheet comparing:

- source
- truth
- stroke renderer
- filled renderer
- auto selector
- naive stroke/filled union
- error image

Main finding:

- filled-only outperformed naive union on the hybrid set
- auto matched filled after lowering the filled area-ratio threshold to `1.25`

## `real-filled-vs-stroke-eval.png`

Diagnostic sheet from real cropped UI icons.

Main finding:

- filled-only is not safe for pure outline icons
- some outline crops produce empty or near-invisible filled masks
- auto must preserve the stroke branch
